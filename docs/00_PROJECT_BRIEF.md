# 00_PROJECT_BRIEF

## 企画名

トレログ

## コンセプト

筋トレを毎日続けられるように、評価と連続記録などをまとめる筋トレ版デュオリンゴ。

かわいい女の子トレーナーとともに筋トレを行う。
トレーナーはメニュー相談、評価、報酬、応援を担当する。

## 体験の核

単なるチェック式記録ではなく、アプリ上で筋トレを開始し、音楽再生や報酬演出によって「今日もやった感」を得られる体験を目指す。

## 初期版で重視すること

- 毎日開きたくなる
- 記録が簡単
- 継続日数が見える
- サボっても復帰しやすい
- トレーナーに応援される
- 音楽を再生して筋トレ開始できる
