# 02_SCREEN_SPEC

## メイン画面

- 画面上部に今日の日付、連続日数、レベル、EXP、今週ランク、休憩チケットを表示する
- トレーナーエリアにはトレーナー画像と通常時のコメントを表示する
- 今日の目標カードには目標スコア、現在スコア、達成率、おすすめ目安を表示する
- 種目ログ入力では種目、記録方式、分または回、きつさを入力する
- 音楽エリアでは音楽ファイル選択と再生を行う
- 「筋トレ開始」ボタンからセッションオーバーレイを開く
- 「種目ログを保存」ボタンから、セッションを使わずに現在入力を保存できる
- 履歴エリアには保存済みの種目ログを表示する

## セッションオーバーレイ

- 筋トレ開始後、通常画面の上に全画面に近いオーバーレイとして表示する
- スマホ縦画面で、筋トレ中にトレーナーコメントとタイマーが見やすい配置にする
- 表示内容は以下とする
- トレーナー画像
- トレーナーコメント
- 現在の種目名
- 記録方式
- 経過時間
- 今日の現在スコア / 目標スコア
- 目標到達までの目安
- セッション中の見込みスコア
- 回数式の場合の今回の回数入力
- 回数式の場合の +1 / +5 / +10 / クリア補助ボタン
- 選択中の音楽ファイル名
- 音楽ファイル選択
- 音楽の再生 / 停止ボタン
- 一時停止 / 再開ボタン
- この種目を記録して終了ボタン
- キャンセルボタン

## セッション操作

- `idle` で筋トレ開始を押すと、セッションオーバーレイを開いて `running` にする
- `running` で一時停止を押すと、タイマーを止めて `paused` にする
- `paused` で再開を押すと、タイマーを再開して `running` にする
- この種目を記録して終了を押すと、既存保存処理で種目ログを保存し、オーバーレイを閉じる
- キャンセルを押すと、保存せずにオーバーレイを閉じる
- 時間式では、タイマーの経過秒数を分に変換した値を保存値にする
- 回数式では、セッションオーバーレイ内の回数入力値を保存値にする
- セッション中の目標到達目安は、保存済みスコアとセッション中の見込みスコアを合算して更新する

## リザルトオーバーレイ

- 種目ログ保存成功後に表示する
- 表示内容は以下の順番とする
- 大見出し
- トレーナー画像
- トレーナーコメント
- 達成バッジ
- 今回の記録: 種目、記録値、きつさ
- 獲得報酬: スコア、EXP
- 今日の進捗: 今日のスコア / 目標スコア、達成率
- 成長: 現在レベル、次のレベルまでのEXP
- OKボタン

## リザルト操作

- 通常記録時は大見出しに「ナイス記録！」を表示する
- 今日の目標達成時は大見出しに「今日の目標達成！」を表示し、紙吹雪を少し多めに出す
- レベルアップ時は大見出しに「レベルアップ！」を表示し、紙吹雪を出す
- 通常記録時は控えめなキラキラを出す
- 今日の目標達成、Lv.UP、休憩チケット獲得は、該当する場合だけバッジ風に表示する
- 休憩チケットを獲得した場合は「休憩チケットをN枚獲得！」を表示する
- OKボタンを押すとリザルトオーバーレイを閉じ、通常画面に戻る
