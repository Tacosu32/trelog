# 02_SCREEN_SPEC

## 全体構成

- 1つのHTML内にホーム、記録、履歴、設定の4画面を持つ
- 下部固定ナビゲーションで、ホーム、記録、履歴、設定を切り替える
- 現在選択中の画面はナビゲーション上で強調表示する
- メインコンテンツ下部には、下部ナビゲーションに隠れない余白を確保する

## ホーム画面

- アプリ名と今日の日付を表示する
- 連続日数、Lv、EXP、今週ランク、休憩チケットを表示する
- レベル進捗バーを表示する
- 今日の目標カードには目標スコア、現在スコア、達成率、おすすめ目安を表示する
- トレーナーエリアにはトレーナー画像と通常時のコメントを表示する
- 「記録を開始する」ボタンを表示する
- 「記録を開始する」ボタンを押すと記録画面へ移動する

## 記録画面

- 種目ログ入力では種目、記録方式、分または回、きつさを入力する
- 記録方式は選択中の種目が対応するものだけ選べる
- プランク、ストレッチを選んだ場合は回数を無効化し、時間へ自動切り替えする
- 記録方式の下に、使える方式を説明する補足文を表示する
- 音楽エリアでは音楽ファイル選択と再生を行う
- 「筋トレ開始」ボタンからセッションオーバーレイを開く
- 「種目ログを保存」ボタンから、セッションを使わずに現在入力を保存できる

## 履歴画面

- 保存済みの種目ログを、年・月・日・種目単位で集約して表示する
- 年、月、日を折りたたみ展開できる
- 日ごとの合計スコア、目標達成表示、記録種目数を表示する
- 種目ごとの合計値、合計スコア、記録件数、平均きつさを表示する

## 設定画面

- 評価レベル設定を表示する
- 評価レベルは、初心者、標準、慣れている、高負荷向けから選択できる
- 評価レベルの説明文として、初心者は軽い運動も評価されやすく、高負荷向けは厳しめに評価されることを表示する
- 開発用デバッグパネルを表示する
- 記録データ削除、状態データ削除、全データ削除を実行できる
- 休憩チケット操作、テスト記録追加、localStorage内容表示を実行できる

## セッションオーバーレイ

- 筋トレ開始後、通常画面の上に全画面に近いオーバーレイとして表示する
- スマホ縦画面で、筋トレ中にトレーナーコメントとタイマーが見やすい配置にする
- 表示内容は以下とする
- トレーナー画像
- トレーナーコメント
- 現在の種目名
- 記録方式
- 経過時間
- 今日の現在スコア / 目標スコア
- 目標到達までの目安
- セッション中の見込みスコア
- 回数式の場合の今回の回数入力
- 回数式の場合の +1 / +5 / +10 / クリア補助ボタン
- 選択中の音楽ファイル名
- 音楽ファイル選択
- 音楽の再生 / 停止ボタン
- 一時停止 / 再開ボタン
- 記録して終了ボタン
- キャンセルボタン

## セッション操作

- `idle` で筋トレ開始を押すと、セッションオーバーレイを開いて `running` にする
- `running` で一時停止を押すと、タイマーを止めて `paused` にする
- `paused` で再開を押すと、タイマーを再開して `running` にする
- `running` 中でも記録して終了を押せる
- `running` 中に記録して終了を押した場合は、その瞬間にタイマーを止めてから既存保存処理で種目ログを保存し、オーバーレイを閉じる
- `paused` 中も記録して終了を押せる
- キャンセルを押すと、保存せずにオーバーレイを閉じる
- 時間式では、タイマーの経過秒数を分に変換した値を保存値にする
- 回数式では、セッションオーバーレイ内の回数入力値を保存値にする
- セッション中の目標到達目安は、保存済みスコアとセッション中の見込みスコアを合算して更新する

## リザルトオーバーレイ

- 種目ログ保存成功後に表示する
- 表示内容は以下の順番とする
- 大見出し
- トレーナー画像
- トレーナーコメント
- 達成バッジ
- 今回の記録: 種目、記録値、きつさ
- 獲得報酬: スコア、EXP
- 今日の進捗: 今日のスコア / 目標スコア、達成率
- 成長: 現在レベル、次のレベルまでのEXP
- OKボタン

## リザルト操作

- 通常記録時は大見出しに「ナイス記録！」を表示する
- 通常記録時は控えめなキラキラを出す
- 今日の目標達成時は大見出しに「今日の目標達成！」を表示し、リザルトオーバーレイ全体に届く紙吹雪を少し多めに出す
- レベルアップ時は大見出しに「レベルアップ！」を表示し、リザルトオーバーレイ全体に届く紙吹雪を出す
- 休憩チケット獲得時も紙吹雪を出す
- 紙吹雪はリザルトオーバーレイ全体を覆うレイヤーで表示する
- 紙吹雪はCSSアニメーションで2〜3秒程度表示し、横方向だけでなく上部から下部まで縦方向に落ちたあと自然に消える
- 今日の目標達成、Lv.UP、休憩チケット獲得は、該当する場合だけバッジ風に表示する
- 休憩チケットを獲得した場合は「休憩チケットをN枚獲得！」を表示する
- OKボタンを押すとリザルトオーバーレイを閉じ、通常画面に戻る

## スマホ操作

- 下部ナビゲーションは画面下部に固定し、親指で切り替えやすい高さにする
- メインコンテンツが下部ナビゲーションに隠れないようにする
- 回数式の +1 / +5 / +10 / クリアなど連打しやすいボタンは、ダブルタップズームが起きにくいように `touch-action: manipulation` を指定する
- ボタンと入力欄はスマホブラウザでズームを誘発しにくいように、基本の文字サイズを16px以上にする
- ボタンは連打時に文字選択が起きにくいようにする

## 履歴表示

- 履歴画面は、年、月、日、種目の階層で折りたたみ表示する
- 初期表示では最新の年、月、日を展開する
- 年行は `2026年` の形式で表示する
- 月行は `6月` の形式で表示する
- 日付行は `2026/06/01　466 / 100pt　達成　3種目` の形式で表示する
- 日付行の達成判定は、その日の合計スコアが目標スコア以上なら `達成`、未満なら `挑戦中` とする
- 種目行は `スクワット　合計120回　180pt　3件　平均きつさ4.0` の形式で表示する
- 時間式の種目行は合計分、回数式の種目行は合計回数を表示する
- `trelog_records` の保存形式は変更せず、画面表示時だけ集計する
