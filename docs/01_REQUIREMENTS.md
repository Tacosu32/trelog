# 01_REQUIREMENTS

## α版の機能要件

### 1. 日付・記録

- 今日の日付を表示する
- 1回の保存は、1つの種目ログとして扱う
- 1日に複数の種目ログを保存できる
- 日ごとの達成判定は、その日に1件以上ログがあれば達成扱いとする
- 過去の履歴を表示する

### 2. 筋トレ入力

- 種目を1つ選択して記録できる
- 記録方式は「時間」または「回数」から選択できる
- 時間式の場合は分単位で入力できる
- 回数式の場合は回数を入力できる
- きつさを1〜5で選択できる
- メモを任意で入力できる
- チェックボックス式のメニューは、「今日のクエスト候補」または「達成済み表示」として残してよい

### 3. 音楽アップロード

- ユーザーが音楽ファイルを選択できる
- 選択した音楽を再生できる
- 音楽再生と筋トレ開始を同じ画面で行える

### 4. 評価

- 種目、記録方式、入力値、きつさからEXPを計算する
- 累計EXPを表示する
- EXPに応じてレベルを表示する
- 今日の評価コメントを表示する

### 5. 継続

- 連続記録日数を計算する
- 未記録日があった場合、凍結チケットを消費して継続を維持する
- 凍結チケットの枚数を表示する

### 6. トレーナー

- トレーナー画像を表示する
- 状況に応じた定型コメントを表示する
- α版では会話機能は実装しない

### 7. 履歴

- 種目ログごとに履歴を表示する
- 履歴には、日付、種目、記録方式、分または回、きつさ、EXPを表示する
- 同じ内容の連続保存を防ぐため、保存後2秒間は保存ボタンを無効化する

## 非機能要件

- スマホ縦画面で見やすい
- 初心者でも理解しやすいコードにする
- データはlocalStorageに保存する
- α版では外部APIを使わない
