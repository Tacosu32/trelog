# 01_REQUIREMENTS

## α版の機能要件

### 1. 日付・記録

- 今日の日付を表示する
- 1日ごとに筋トレ記録を保存する
- 過去の履歴を表示する

### 2. 筋トレ入力

- チェックボックスでメニューを選択できる
- 運動時間を分単位で入力できる
- きつさを1〜5で選択できる
- メモを任意で入力できる

### 3. 音楽アップロード

- ユーザーが音楽ファイルを選択できる
- 選択した音楽を再生できる
- 音楽再生と筋トレ開始を同じ画面で行える

### 4. 評価

- メニュー数、時間、きつさからEXPを計算する
- 累計EXPを表示する
- EXPに応じてレベルを表示する
- 今日の評価コメントを表示する

### 5. 継続

- 連続記録日数を計算する
- 未記録日があった場合、凍結チケットを消費して継続を維持する
- 凍結チケットの枚数を表示する

### 6. トレーナー

- トレーナー画像を表示する
- 状況に応じた定型コメントを表示する
- α版では会話機能は実装しない

### 7. 履歴

- 日付、メニュー、時間、EXPを一覧表示する

## 非機能要件

- スマホ縦画面で見やすい
- 初心者でも理解しやすいコードにする
- データはlocalStorageに保存する
- α版では外部APIを使わない
