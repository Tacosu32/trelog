# AGENTS.md

## Project

このプロジェクトは「トレログ」です。

トレログは、筋トレを毎日続けられるように、記録・評価・連続日数・報酬・トレーナーキャラによる応援を提供する、筋トレ版デュオリンゴ風Webアプリです。

## Concept

かわいい女の子トレーナーと一緒に筋トレを行います。
トレーナーはメニュー相談、評価、報酬、応援を担当します。

単なるチェック式の筋トレ日記ではなく、音楽アップロード・再生・筋トレ開始ボタンなどにより、アプリ上で筋トレを始める体験を作ります。

## Current Goal

まずはHTML/CSS/JavaScriptでα版を作成します。

React、TypeScript、バックエンド、LLM API、カメラAI、スマホアプリ化は後回しにします。

## Development Rules

- 仕様は docs/ 配下のMarkdownを参照してください。
- 大きな変更を一度に行わないでください。
- 既存仕様を変える場合は docs/05_DECISIONS.md に理由を追記してください。
- 関数を追加・変更した場合は docs/04_FUNCTIONS.md を更新してください。
- APIキーや個人情報をコードに書かないでください。
- 初心者が理解できるように、複雑な実装は避けてください。
- まずは動くものを優先し、過度な最適化はしないでください。
- スマホ縦画面で使いやすいUIを優先してください。

## MVP Scope

α版で作る機能は以下です。

- 今日の日付表示
- チェックボックス式筋トレメニュー
- 運動時間入力
- きつさ入力
- 音楽ファイルアップロード
- 音楽再生と筋トレ開始
- 記録保存
- EXP計算
- レベル表示
- 継続日数表示
- 凍結チケット
- 履歴表示
- 簡易トレーナーコメント
- スマホ向け縦長UI

## Out of Scope for Alpha

α版では以下を実装しません。

- LLM API連携
- 自由会話
- カメラ姿勢推定
- スマホネイティブアプリ化
- Push通知
- ウィジェット
- 3Dモデル表示
- クラウド同期
