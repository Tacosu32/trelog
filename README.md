# トレログ

トレログは、筋トレを毎日続けるための筋トレ版デュオリンゴ風Webアプリです。

## コンセプト

かわいい女の子トレーナーと一緒に筋トレを行い、記録・評価・連続日数・報酬によって継続を支援します。

## α版の目的

まずはブラウザで動くWebアプリとして、筋トレ記録、EXP、継続日数、凍結チケット、音楽再生、簡易トレーナー応援を実装します。

## 技術構成

- HTML
- CSS
- JavaScript
- localStorage
